# Sistema Inteligente de Detección y Conteo de Objetos
## Descripción
Sistema de visión artificial que detecta y cuenta personas u objetos en tiempo real usando una cámara web.

## Tecnologías
- Python
- FastAPI
- OpenCV
- YOLOv8
- HTML, CSS, JavaScript

## Funcionalidades
- Detección en tiempo real
- Conteo automático de objetos
- Streaming de cámara web
- Interfaz web sencilla

## Ejecución

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
